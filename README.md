# 42_libft_tester
A simple and step-by-step tester to C 42 Library.

# How to use it ?
You need to clone this tester in your own Libft repository, go in the tester directory, and run tester.sh with an option:\
`./tester.sh mand`\
`./tester.sh bonus`\
`./tester.sh all`
