# 42 Libft Tester
A simple step-by-step tester to C 42 Library.

# How to use it ?
- Clone the repository in your Libft directory, go in the tester directory, and run `tester.sh` with an argument:
```
./tester.sh mand
./tester.sh bonus
./tester.sh all
```
- Press Enter to perform each test.
